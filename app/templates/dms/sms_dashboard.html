India SMS Setup Guide

✅ آسان طریقہ — Fast2SMS (مفت 200 SMS)
  1. fast2sms.com پر account بنائیں
  2. Dashboard سے API Key لیں
  3. .env فائل میں لکھیں:
SMS_PROVIDER=fast2sms
SMS_API_KEY=your_api_key_here
SMS_SENDER_ID=FSTSMS
🚀 Professional — MSG91 (paid، زیادہ reliable)
  1. msg91.com پر account بنائیں
  2. Sender ID approve کروائیں (6 حروف)
  3. .env فائل میں لکھیں:
SMS_PROVIDER=msg91
SMS_API_KEY=your_authkey_here
SMS_SENDER_ID=DMSSYS
SMS_ROUTE=4
Route 4 = Transactional (OTP/alerts)، Route 1 = Promotional
.env فائل تبدیل کرنے کے بعد server restart کریں
